# zillow-data-cleaning-pipeline
Automated data cleaning pipeline for Zillow real estate dataset using Python and Pandas. Handles missing values, duplicates, and data inconsistencies.

# 🏠 Zillow Real Estate Data Cleaning Pipeline

## 📌 Overview
This project is a fully automated data cleaning pipeline built using Python.  
It processes raw Zillow real estate datasets and generates clean, structured CSV files ready for analysis.

---

## 🚀 Key Highlights
- Processed **7 real-world datasets**
- Handled **360,000+ missing values**
- Removed **28,000+ duplicate records**
- Built a **modular and automated pipeline**
- Delivered **analysis-ready cleaned data**

## ⚙️ Features
- Automated data cleaning pipeline
- Missing value handling
- Duplicate removal
- Outlier detection and handling
- Modular script-based architecture
- Batch processing of multiple CSV files

---

## 🧠 Pipeline Workflow
1. Load raw datasets  
2. Handle missing values  
3. Remove duplicate records  
4. Detect and handle outliers  
5. Export cleaned datasets
